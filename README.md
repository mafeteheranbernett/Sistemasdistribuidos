# Sistemasdistribuidos
1. ¿Qué ventajas encuentras en trabajar directamente en la nube con GitHub?: El hecho de poder hacerlo en cualqioer lugar, de no estar dependiendo de una maquina o computador, y poder buscar mas informacion por si uno se pierde son grandes ventajas
2. ¿Qué desafíos tuviste al crear el sitio sin usar herramientas locales?: Por ejemplo si no se guarda correctamente se tiene que rehacer, el hecho tambien de no poder ver los cabios al intante como para saber si uno va bien o no, es un gran problema
3. ¿Cómo podrías mejorar este sitio web en el futuro?: haciendolo un poco mas intuitivo y definitivamente tratar de bajar el tiempo de espera para poder guiarse bien de cualquier cambio que se haga  
4. ¿Qué aprendiste sobre Sistemas Distribuidos al crear este sitio?: El poder trabajar en la nube con diferentes herramientas es una muestra de los sistemas distribuidos ya que directamente los estamos manejado
